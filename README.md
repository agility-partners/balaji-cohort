# crypto-watch-website
A Cryptocurrency Watchlist website built using React, TypeScript, Tailwind CSS, Next.js, and Playwright

## Usage

1. Clone this repository:
    ```
    git clone https://github.com/agility-partners/balaji-cohort.git .
    cd crypto-watchlist
    ```

2. Run website
    ```
    npm run dev
    ```

3. Navigate to `http://localhost:3000` on a browser
